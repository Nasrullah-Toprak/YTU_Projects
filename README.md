## 📈 Optimization Techniques for Answer Quality Assessment
A machine learning and optimization project focused on evaluating the quality of answers to natural language questions.

- Generated question-answer datasets containing good and poor responses
- Semantic embeddings extracted using transformer-based language models
- Binary classification using a tanh-based neural model
- Comparison of Gradient Descent (GD), Stochastic Gradient Descent (SGD), and Adam optimizers
- Training and testing performance evaluation
- Runtime, loss, accuracy, and convergence analysis
- Optimization trajectory visualization using t-SNE

**Topics:** Machine Learning, Optimization Algorithms, NLP, Data Visualization

**Tools:** MATLAB, Transformer Embeddings, t-SNE

## 🎵 Digital Signal and Image Processing Using MATLAB
A collection of MATLAB projects covering signal processing, audio processing, convolution, and image restoration techniques.

### Discrete-Time Convolution and Audio Echo System
- Custom convolution implementation from scratch
- Validation against MATLAB's built-in conv() function
- Mean Squared Error (MSE) performance analysis
- Audio recording, playback, and echo generation
- LTI system modeling and impulse response analysis
- Parametric study of echo coefficients and delay parameters

### Frequency-Domain Image Denoising
- Image preprocessing and grayscale conversion
- 2D Fourier Transform (FFT) analysis
- Automatic periodic noise frequency detection
- Notch filtering in the frequency domain
- Inverse FFT image reconstruction
- Visualization of frequency spectra and filtering effects

**Topics:** DSP, Convolution, Fourier Transform, Audio Processing, Image Processing

**Tools:** MATLAB, FFT, Signal Analysis
