# 🖥️ RISC-V Computer Architecture Projects

A collection of projects completed as part of the **Computer Organization and Computer Architecture** coursework. These projects focus on hardware design, processor implementation, assembly programming, timing analysis, and performance evaluation using the **RISC-V Instruction Set Architecture (ISA)**.

The repository covers multiple abstraction levels of computer systems, ranging from gate-level digital logic design to complete processor implementation and low-level assembly software development.

---

## 📂 Projects Included

### 1. High-Performance Structural RISC-V ALU and Datapath Design

Design and implementation of a 32-bit RISC-V ALU using structural Verilog.

**Highlights:**
- Gate-level implementation using basic logic gates
- Support for AND, XOR, ADD, SUB, and SLT instructions
- High-speed adder architecture
- Datapath integration with register file
- Maximum frequency (Fmax) analysis
- Hold-time and setup-time verification
- Critical path optimization
- Timing-aware hardware design

**Topics:**
- Digital Logic Design
- Structural Verilog
- ALU Architecture
- Timing Analysis
- Hardware Optimization

---

### 2. Single-Cycle RISC-V Processor Implementation and ISA Extensions

Development of a complete single-cycle RISC-V processor capable of executing a subset of the RV32I instruction set and supporting custom ISA extensions.

**Highlights:**
- Complete single-cycle processor design
- Instruction fetch, decode, execute, memory, and write-back stages
- Support for arithmetic, logical, memory, branch, and jump instructions
- ISA extension for arithmetic shift instructions
- ISA extension for halfword memory operations
- Modular processor architecture
- Verification through custom testbenches and waveform analysis

**Topics:**
- Processor Design
- Control Unit Design
- Datapath Architecture
- Instruction Decoding
- ISA Extensions

---

### 3. RISC-V Assembly Programming and Cycle Analysis

Implementation and analysis of a RISC-V assembly application executed on a custom processor architecture.

**Highlights:**
- Assembly-level software development
- Machine code generation
- Bit manipulation algorithms
- Memory access and register optimization
- Execution cycle analysis
- Performance evaluation
- Hardware-software integration
- Verification using GTKWave simulations

**Topics:**
- RISC-V Assembly
- Machine Language
- Processor Execution
- Cycle Counting
- Performance Analysis

## 📈 Project Progression

The projects are organized in a progressive manner:

**Project 1**
→ Design of a high-performance ALU and optimized datapath.

**Project 2**
→ Construction of a complete single-cycle RISC-V processor using the previously developed components.

**Project 3**
→ Development and execution of assembly software on the implemented processor architecture.

Together, these projects demonstrate the complete workflow from low-level digital hardware design to software execution on a custom processor.
